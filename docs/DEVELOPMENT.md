# Development Guide

This guide covers local setup and common development commands for Gravatar Worker. For contribution expectations and PR review guidance, see [`CONTRIBUTING.md`](../CONTRIBUTING.md).

## Overview

The project uses:

- Node.js from `.node-version`
- pnpm from `package.json`
- Cloudflare Workers
- Wrangler
- Hono
- Vite

Do not commit secrets or local environment files.

## Prerequisites

Check Node.js:

```bash
node --version
```

Enable pnpm through Corepack if needed:

```bash
corepack enable
pnpm --version
```

## Install Dependencies

From the repository root:

```bash
pnpm install --frozen-lockfile
```

Use `pnpm install` without `--frozen-lockfile` only when intentionally updating the lockfile.

## Local Variables

For local Worker variables:

```bash
cp .dev.vars.example .dev.vars
```

Then edit `.dev.vars` with local-only values such as `ME_EMAIL`.

The deployed Worker uses variables from `wrangler.jsonc` unless you explicitly set `"keep_vars": true`.

## Cloudflare Login

Some Wrangler commands require a Cloudflare session:

```bash
pnpm exec wrangler login
```

Use an account with access to the target Cloudflare Worker when deploying.

## Start Development Server

```bash
pnpm dev
```

Vite prints the local URL, usually:

```text
http://localhost:5173/
```

## Build

```bash
pnpm run build
```

If Wrangler cannot write logs in a sandboxed environment, set the log path inside the repository:

```bash
WRANGLER_LOG_PATH=.wrangler/logs pnpm run build
```

## Lint

```bash
pnpm run lint
```

To apply safe lint fixes:

```bash
pnpm run lint:fix
```

## Regenerate Cloudflare Types

After adding or changing Worker variables or bindings:

```bash
pnpm run cf-typegen
```

Review `worker-configuration.d.ts` before committing. It is generated by Wrangler and may include upstream runtime type changes.

## Deploy

```bash
pnpm run deploy
```

Before deploying, confirm:

- required variables are in `wrangler.jsonc` or intentionally managed through the Cloudflare Dashboard
- `.dev.vars` does not contain secrets or values that need to be committed
- `pnpm run lint` passes
- `pnpm run build` passes

## Useful Scripts

| Command               | Purpose                                    |
| --------------------- | ------------------------------------------ |
| `pnpm dev`            | Start the local development server         |
| `pnpm run build`      | Build the Worker and client assets         |
| `pnpm run deploy`     | Build and deploy with Wrangler             |
| `pnpm run lint`       | Run ESLint                                 |
| `pnpm run lint:fix`   | Run ESLint and apply safe fixes            |
| `pnpm run cf-typegen` | Regenerate Cloudflare Worker binding types |
